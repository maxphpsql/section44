== Description ==

Import the Envo Business demo layouts with one click.
This plugin requires the [Envo Business](https://envothemes.com/envo-business/) theme and [KingComposer](https://wordpress.org/plugins/kingcomposer/) plugin to be installed.
Visit the [documentation page](https://envothemes.com/envo-business/theme-documentation/#importing-the-sample-data) for more instructions.

== Installation ==

1. Upload `envothemes-importer-kingcomposer` to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

= I installed the plugin but it does not work =
